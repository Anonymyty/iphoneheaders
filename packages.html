About the packages on my Cydia repo

Curl and Git are both re-packaged versions of old versions to fix conflicts created by the newer packages on http://cydia.radare.org/

DevRepos add two repos that have a lot more and some newer terminal commands. DevCommands will then install a bunch of helpful commands

mach_portal icon includes a couple artworks created by Reddit users to theme the mach_portal icon on the iOS 10 Yalu jailbreaks.

EW39.iWidget is a highly modified version of Evelyn's EW39.iWidget. The track name and artist are shown above the dock. Currently only optimized for iPhone 6 resolution.

If you'd like me to host one of your packages, feel free to contact me
